📦 JS / CSS 仓库 API

✓ 永久存储层 ✓ CORS 友好 ✓ 即开即用 ✓ 支持任意资源

本站点作为您项目的静态资源仓库，专用于托管 JavaScript 与 CSS 文件。
您可以自由上传/替换 .js 或 .css 文件，并通过 URL 直接引用。 

<!-- 在您的网站中引入本仓库的 CSS 示例 -->
<link rel="stylesheet" href="https://res.xscnet.cn/style.css">

<!-- 引入本仓库的 JavaScript 示例 -->
<script src="https://res.xscnet.cn/script.js"></script>

<!-- 也可以使用带版本或参数的路径（需要您自行规划文件）-->
<script src="https://res.xscnet.cn/lib/utils.js"></script>

✨ 作为“仓库 API” 提供的特性：

  • ✅ 纯静态托管，无后端依赖 —— 完美适配 GitHub Pages / Vercel
  • ✅ 支持任意 JS/CSS 文件直接引用，如同使用 CDN
  • ✅ 内置了针对 JSONP 和模块化脚本的兼容结构（不限制请求方式）
  • ✅ 可同时存放多个库文件，通过目录结构进行版本管理
  • ✅ 无内容限制，只作为资产管道，不干扰业务逻辑

⚡ 动态 API 演示 (Query 路由) —— 纯粹概念展示，不影响文件系统:

➤ 加载中...

?get=example.css 或 ?type=js&name=theme 演示: 本页面可以通过查询参数返回示例代码片段（仅为演示API风格）。 实际使用时，请直接通过静态文件路径获取资源。

🚀 部署说明：将您的 JS/CSS 文件放在同一仓库目录下，提交并部署到 GitHub Pages 或 Vercel 即可获得永久链接。
本页面自身不包含任何业务UI，仅为仓库信息卡。您也可以删除此说明卡片，仅保留资源文件结构。
